🌋 Volcano Plot Visualizer

Upload your differential expression analysis results to create interactive volcano plots - No installation required!

📚 Help & Status

🎯 Purpose

This tool generates interactive volcano plots for RNA-seq differential expression analysis results. It supports various analysis tools and provides customizable visualization options.

📋 Supported Formats

  • DESeq2: gene, log2FoldChange, pvalue, padj
  • edgeR: gene, logFC, PValue, FDR
  • limma: gene, logFC, P.Value, adj.P.Val
  • metabolomics: metabolite, log2FoldChange, pvalue, padj
  • other: gene, log2FoldChange, pvalue, padj

📊 Features

  • Interactive hover tooltips
  • Click to add gene/metabolite into the plot
  • Customizable cutoffs and colors
  • Customizable x/y labels
  • Customizable legend title
  • Export plots in PNG, SVG, PDF
  • Real-time plot updates

📝 How to Prepare Your Data

  • File Format: Tab-delimited text file (.txt, .tsv, .csv)
  • Required Columns: Four columns (gene, log2foldchange, pvalue, padj)

💡 Creating Tab-Delimited Files

Using Excel:

  1. Open your data in Excel
  2. "File" → "Save As"
  3. "Text (Tab delimited) (*.txt)" from the file type dropdown
  4. Click "Save"

Using WPS Office:

  1. Open your data in WPS Spreadsheets
  2. "File" → "Save As"
  3. "Text (Tab delimited) (*.txt)" from the format options
  4. Click "Save"
